# Database Keys: Primary Key vs Foreign Key
| Aspect            | Primary Key (PK)                     | Foreign Key (FK)                         |
|-------------------|--------------------------------------|------------------------------------------|
| Purpose           | Uniquely identifies a row             | Links to a primary key in another table  |
| Uniqueness        | Must be unique                        | Can repeat                               |
| Null values       | Not allowed                           | Allowed (unless constrained)             |
| Table scope       | Defined in its own table              | References another table                 |
| Data integrity    | Ensures entity integrity              | Ensures referential integrity            |
